# Hero images (optional)

Drop ONE landscape photo per industry here to upgrade demos from the gradient hero to a
real photo. Filenames must match the preset keys:

- `home_services.jpg`
- `dental.jpg`
- `medical.jpg`
- `wellness.jpg`
- `legal.jpg`
- `accounting.jpg`
- `real_estate.jpg`
- `insurance.jpg`
- `service.jpg`  (generic fallback)

Recommended: ~1600x1000, < 300 KB, licensed stock or your own. Missing files are fine:
the page falls back to the gradient hero automatically. Never use a prospect's own
scraped photos here; the demo says the image is a sample.
